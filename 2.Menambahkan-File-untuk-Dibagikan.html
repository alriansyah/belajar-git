Peranan Git :
        Ada 3 langkah pada pengembangan tim.
        1. Buat Perubahan ke file
        2. Persiapan untuk membagikan Perubahan
        3. Bagikan perubahan

        Lihat pada step 2 & 3. Kamu perlu untuk membagikan code yang kamu tulis dengan orang lain untuk melakukan pengembangan tim. 
        Itu gunanya Git.

        Memilih File untuk Dibagikan :
        NK : Aku baru membuat file index.html! Jadi, bagaimana aku bisa membagikannya?
        GD : Pertama, kamu perlu memilih file-file yang ingin kamu bagikan.
             Untuk memilih sebuah file, gunakan command git add file_name.
        
             Lakukan : git add file_name
